le jeu du pendu :
quand le program est lancé, le program doit attendre un input()
au début l'input 1 sera le nombre de lettres dans le mot et le second input sera le nombre d'essai que la machine aura
ensuite un fois lançé le program ne s'arréte pas tant qu'il n'as pas trouvé ou n'as plus assez de vie
quand il n'as plus assez de vie le jeu affiche le pendu si il gagne il affiche le pendu qui court
a chaque tour le pendu donne une lettre, si la lettre est dedans le player doit entrer 'yes [number]' pour dire l'emplacement de la lettre dans le mot sinon le player doit mettre 'non'
lorsque le player met non, la machine perd une vie
lorsqu'il met oui la machine affiche la lettre en plus dans le mot
le mot doit etre afficher en permanence, si des lettres ne sont pas découvertes, la machine affiche "_" a la place de la lettre
